# Screenshot Maker Example

This example demonstrates all the features of the `s_screenshot` package.

## Features Demonstrated

1. **Base64 Capture** - Capture and display as base64 string
2. **Bytes Capture** - Capture as raw bytes (Uint8List)
3. **File Capture** - Capture and save directly to file
4. **Delayed Capture** - Capture after a delay (useful for animations)

## Running the Example

```bash
flutter pub get
flutter run
```
